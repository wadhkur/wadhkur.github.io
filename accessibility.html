Accessibility Statement

This site is intended to be usable by everyone. The design follows WCAG 2.2 level AA guidance where practical for a static site.

Standards

The site uses semantic HTML, clear focus states, high-contrast text, and supports keyboard navigation. Animations respect the user’s prefers-reduced-motion setting.

Testing

Pages are reviewed for keyboard access, logical heading order, and form-free navigation appropriate to static content.

Known limitations

Embedded media and downloadable documents are not present. If new formats are added, alternative text or transcripts will be provided.

Feedback

If you experience barriers, please inform the clinic team. Feedback will be reviewed and addressed where feasible.
